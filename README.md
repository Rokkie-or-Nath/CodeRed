# 🚨 PulsePoint

> "Fast help when every second matters."
PulsePoint is a web-based Emergency Locator application designed to help users quickly find nearby hospitals during urgent situations.
The app uses real-time geolocation and an interactive map to display nearby hospitals. With a single click on the EMERGENCY button, users can instantly highlight the nearest hospital and respond faster in critical moments.

---

## 🌟 Features

- 📍 Real-time user location detection (Geolocation API)
- 🗺️ Interactive map integration (Mapbox / Google Maps)
- 🏥 Nearby hospital search with map markers
- 🚨 Emergency button to highlight the nearest hospital
- 📱 Responsive design for desktop and mobile
- 🎨 Clean and modern UI

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Mapbox API / Google Maps API
